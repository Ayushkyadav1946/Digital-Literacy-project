# 🧠 Digital Literacy Awareness Quiz

> A simple interactive quiz designed to test and improve awareness about **online safety and digital literacy**.

---

## 📌 About the Project

This project is a **Google Forms-based quiz** that helps users understand how to stay safe online.

It focuses on real-world scenarios like:
- Suspicious emails 📧  
- Unsafe links 🔗  
- Website security 🔒  
- VPN usage 🌐  

---

## 📝 Quiz Highlights

The quiz includes questions such as:

- 🎣 What is phishing?  
- 📩 What to do if you receive a fake bank email?  
- 🔍 How to verify links before clicking?  
- 🔒 What does the padlock icon mean in a browser?  
- 🌐 What is a VPN used for?  

👉 These questions are designed to make users **think before they act online**.

---

## 📊 Results & Performance

- Responses are recorded automatically using **Google Sheets**
- Instant feedback is provided after submission  
- Example result: ✅ **5/5 score achieved**

This shows that the quiz is **clear, effective, and easy to understand**.

---

## 🎯 Key Learning Outcomes

After taking this quiz, users can:

- Identify phishing attempts  
- Avoid clicking harmful links  
- Understand secure websites  
- Use basic cybersecurity practices
